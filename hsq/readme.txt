Higher Subleq (HSQ)

C compiler for Subleq

online 		system building HSQ as CGI script (HSQ online interface)
cop		shell script to run automatic tests
hsq.cpp		HSQ compiler
mant.fcl	a few manual tests (fcl archive)
mkhtml.sh	generates html pages with test results
sqasm.cpp	(deprecated) Subleq assembler
sqrun.cpp	(deprecated) Subleq emulator
sts.fcl		collection of automatic tests (fcl archive)

sqasm and sqrun are deprecated because HSQ has taken over their functionality
